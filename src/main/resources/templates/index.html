This is a server build in spring boot for hack debug API call

Install steps

Step 1: Generate self-generated root certificate

Generate self-generated root certificate from here

I already generated the root certificate in src/main/resources/cert/root.pem, you can use it directly.

If you want regenerate a new, just run src/main/resources/cert/1.gen-root.sh.

Step 2: Install self-generated root certificate to your system

How to install self-generated root certificate into your device from here

Step 3: Generate domain certificate for your domain

Add domain name you want hack into src/main/resources/cert/domain.ext e.g. DNS.2 = *.local.dev

Run src/main/resources/cert/2.gen-cert.sh to generate certificate

Step 4: Add your hack logic

See hack demo in HackController.java

Step 5: Build & launch app

Enter into project root path and run ./gradlew bootRun.

If you are running on 80 or 443 port you need sudo permissions.

Step 6: Bind host and visit

Bind hosts 127.0.0.1 hello.local.dev. See how to bind host here

Visit https://hello.local.dev/imghp
